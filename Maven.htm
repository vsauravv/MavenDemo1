What is Maven?

Maven is an automation and management tool developed by Apache Software Foundation. It was initially released on 13th July 2004. In Yiddish language, the meaning of Maven is "accumulator of knowledge". It is written in Java language and used to build and manage projects written in C#, Ruby, Scala and other languages. It allows the developer to create projects using Project Object Model and Plugins. It helps to build projects, dependency and documentation. Its developed process is very similar to ANT. However, it is much advanced than ANT. Maven is also able to build any number of projects into desired output such as jar, war, metadata.

    How can Maven benefit my development process?

    Maven helps the developer to create a java-based project more easily. Accessibility of new feature created or added in Maven can be easily added to a project in Maven configuration. It increases the performance of project and building process.
